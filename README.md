# Ballon d'Or Fairness – Tableau Visualization

## Overview
This project presents an interactive Tableau dashboard analysing the fairness of
Ballon d'Or awards by comparing player voting results with performance statistics,
team trophies, and FIFA player ratings.

## Files Included
- `BallonDor_Visualization.twbx` – Packaged Tableau workbook (all data embedded)
- `Attributes.xlsx` – FIFA player attributes dataset
- `GAs.xlsx` – Goals and assists dataset
- `testSet.xlsx` – Ballon d'Or voting and trophies data
- `A3_GhareMukul_Video.mp4` – Short demonstration of the visualization

## Software Requirements
- Tableau Desktop OR Tableau Reader (free)

## How to Run
1. Download `BallonDor_Visualization.twbx`
2. Open it using Tableau Desktop or Tableau Reader
3. Use the Year filter to explore different seasons
4. Click on player bars to filter other views
